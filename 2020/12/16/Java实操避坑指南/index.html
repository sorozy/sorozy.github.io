Java 空指针和异常

出现空指针的情况

（1）调用了空对象的实例方法

（2）访问了空对象的属性

（3）当数组是一个空对象的时候，取它的长度

（4）null当作Throwble的值

（5）方法的返回值是null，调用方法直接去使用

避免空指针异常的方法

（1）使用之前要初始化;使用前需要校验、判断是否为null

（2）尽量避免在函数中返回null
